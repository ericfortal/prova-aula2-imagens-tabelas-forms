Prova Imagens / Tabelas e Forms

Nessa atividade iremos criar um formulário em uma página web aplicando uma série de tipos de formulário onde o usuário poderá em um campo específico, escrever um texto, inserir senha, e-mail, teremos aplicação de checkboxes onde o usuário poderá selecionar mais de uma opção, teremos radiobutton com 2 opções onde o usuário poderá selecionar apenas uma opção por vez, teremos botão para envio de informações do formulário e por último, um botão de reset que limpará todo o formulário para que o usuário possa começar o preenchimento novamente.

Execução da prova abaixo:

Cadastro completo de usuário:







  • Escolha sua opções de preferência

Comentários gerais sobre a execução da prova acima

  • Na etapa de criação de uma lista com 5 checkboxes, apesar de não ter ficado claro no pré-requisito da prova, como forma de organizar melhor criei uma lista não ordenada, com outra lista aninhada não ordenada contendo as 5 opções conforme pedido na questão.

  • Na etapa de criação do checkbox por padrão o html, coloca a caixa de seleção após a descrição da caixa, acreditando ficar mais estético, posicionei a caixa antes da descrição colocando o input dentro de label, a mesma lógica foi utilizada para o radio.

  • Antes de utilizar o radio, foi colocado um parágrafo "p" utilizando a tag "b" para evidenciar uma pergunta que levaria o usuário a responder por meio de uma marcação única utilizando o atributo type definido como "radio".
